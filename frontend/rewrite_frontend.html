Qwen LoRA Fine-Tune Tutorial

This page turns the repo into a guided demo: run the FastAPI service, send text to your LoRA adapter, and compare how the fine-tuned model rewrites technical prose.

Base API `POST /rewrite` on `http://127.0.0.1:8000`
Use Case Academic-style rewriting with a local Qwen adapter
Back End FastAPI + `mlx_lm.generate` + adapter path from env vars
1

Start the API

Run the local server so this page can call your fine-tuned model.

uvicorn app:app --reload
2

Keep your adapter path correct

The app reads `MLX_ADAPTER_PATH` and `MLX_BASE_MODEL` from the environment. Change them when you switch checkpoints.

3

Paste text and compare the rewrite

Use the playground below to see how the fine-tune changes tone, precision, and phrasing.
